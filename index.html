Privacy Policy

Last Updated: September 16, 2025

This Privacy Policy outlines how the n8n LinkedIn Automation application ("the Application") handles data in compliance with applicable regulations. The Application is a personal automation tool designed to aggregate publicly available Hong Kong news, generate social media content using a local AI model, and post to LinkedIn to raise awareness on relevant topics.

1. Data Collection

The Application collects publicly available news headlines from sources such as the South China Morning Post or Hong Kong Free Press. No personal data is collected from these sources. The Application does not store or process user data beyond temporary processing for content generation.

2. Data Usage

Collected news headlines are processed locally using an open-source AI model (e.g., phi3 via Ollama) to generate LinkedIn posts. These posts are shared via LinkedIn’s API, authenticated with OAuth2, using permissions granted by the user’s LinkedIn account. Data is not shared with third parties.

3. Data Storage and Security

All processing occurs on a local server. No data is stored persistently or transmitted to external servers beyond LinkedIn’s API for posting. Reasonable measures are taken to secure the local environment, including access controls and encryption where applicable.

4. User Rights

As no personal data is collected or stored, there are no user data records to access, modify, or delete. For concerns about LinkedIn posts, users may contact the application owner to manage content.

6. Changes to This Policy

This policy may be updated to reflect changes in the Application’s functionality. Updates will be posted on this page with a revised "Last Updated" date.
